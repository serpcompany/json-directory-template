# Starter Setup Checklist

Use this quick checklist when configuring one of the checked-in starter sites.

## Core steps

1. Install the repo dependencies with `nvm use` and `pnpm install`
2. Create or update `sites/<id>/site-config.ts`
3. Add any site-owned extras in `sites/<id>/site-content.ts`
4. Populate or generate `data/listings.json`
5. Run `pnpm dev` to review the site locally
6. Run `pnpm build:site -- --site <id>` before deploying

## Recommended follow-up

- Review the public docs in `/docs`
- Update the About and legal content
- Verify listing detail pages and submit flow
- Check the final artifact under `dist/sites/<id>`
