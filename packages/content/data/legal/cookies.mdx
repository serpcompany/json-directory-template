# Cookie Policy

**Last updated: April 5, 2026**

This Cookie Policy explains how this directory site ("we," "us," or "our") uses cookies and similar tracking technologies when you visit the public site and any configured extensions of the starter.

## 1. What Are Cookies

Cookies are small text files that are stored on your device (computer, tablet, or mobile) when you visit a website. They are widely used to make websites work more efficiently and to provide information to website owners.

### 1.1 Types of Cookies We Use

- **Session Cookies**: Temporary cookies that are deleted when you close your browser
- **Persistent Cookies**: Cookies that remain on your device for a set period or until you delete them
- **First-Party Cookies**: Cookies set by our website directly
- **Third-Party Cookies**: Cookies set by external services we use

## 2. How We Use Cookies

### 2.1 Essential Cookies

Most starter sites do not set login or analytics cookies by default. If you keep the default
feature flags and do not configure authentication or analytics, the public site works without
those cookies.

If a site owner enables additional integrations later, that site may add cookies for those
features and should update this policy to match the real deployment.

These cookies are only relevant when optional authentication is enabled:

| Cookie Name | Purpose | Duration | Type |
| ----------- | ------- | -------- | ---- |
| `next-auth.*` | Optional GitHub sign-in session state | Session/default auth lifetime | Essential |

### 2.2 Analytics Cookies

This starter does not ship analytics cookies by default.

If a site owner later enables analytics, they should update this section to document the actual
provider and consent behavior.

### 2.3 Local Storage (Not Cookies)

We use browser local storage (not cookies) to enhance your experience:

| Stored Preference | Purpose | Duration | Type |
| ----------------- | ------- | -------- | ---- |
| `llms-txt-hub-favorites` | Store your saved listings locally | Until manually cleared | Local Storage |
| `websites-show-all` | Remember whether the directory expands all listings | Until manually cleared | Local Storage |
| `websites-sort-by` | Remember the default directory sort order | Until manually cleared | Local Storage |
| `category-sort-by` | Remember category-page sort order | Until manually cleared | Local Storage |
| `recentSearches` | Store recent search history for autocomplete | Until manually cleared | Local Storage |

**Note**: Local storage data stays on your device and is not sent to our servers. This data persists until you clear your browser data or remove it manually.

## 3. Cookie Consent

### 3.1 Essential Cookies
If optional authentication is enabled for your site, the auth provider may set essential session
cookies while a user signs in or stays logged in.

The default starter configuration does not enable sign-in.

### 3.2 Non-Essential Cookies
The default starter does not set non-essential analytics or marketing cookies.

If a site owner adds them later, they should document the real provider and consent behavior here.

## 4. Managing Your Cookie Preferences

### 4.1 Browser Settings

You can control cookies through your browser settings:

- **Chrome**: Settings > Privacy and Security > Cookies and other site data
- **Firefox**: Settings > Privacy & Security > Cookies and Site Data
- **Safari**: Preferences > Privacy > Cookies and website data
- **Edge**: Settings > Privacy, search, and services > Cookies and site permissions

### 4.2 Cookie Categories

- **Strictly Necessary**: Cannot be disabled as they are essential for website functionality
- **Analytics**: Can be disabled; will not affect website functionality
- **Functional**: Can be disabled; may affect some website features

### 4.3 Impact of Disabling Cookies

| Cookie Type | Impact if Disabled |
| ----------- | ------------------ |
| Essential (`next-auth.*`, if enabled) | Cannot log in or maintain sessions |
| Analytics | No impact on functionality on the default starter |

### 4.4 Impact of Clearing Local Storage

| Storage Type | Impact if Cleared |
| ------------ | ----------------- |
| Favorites | Will lose all saved favorite listings |
| Sort/Filter Preferences | Will reset to default sort and filter settings |
| Recent Searches | Will lose search history and autocomplete suggestions |

## 7. Cookie Security
