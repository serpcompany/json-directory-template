# Task: scripts/submissions.ts — admin CLI for list, verify, publish

## Context

Directory owners need a way to see incoming submissions and publish verified ones to `data/listings.json` without a web UI. A simple CLI script covers the MVP. Runs locally with `pnpm submissions <command>`.

Can start development as soon as task 58 (data contracts) is done.

## Implementation

**Create `scripts/submissions.ts`:**

Commands:
- `list-pending` — table view of all pending submissions.
- `list-verified` — table view of all verified submissions, with published status.
- `publish <token>` — promotes a verified submission to `data/listings.json`.

```ts
// Usage:
//   pnpm submissions list-pending
//   pnpm submissions list-verified
//   pnpm submissions publish abc123-def456

const command = process.argv[2]
const arg = process.argv[3]

switch (command) {
  case 'list-pending': return listPending()
  case 'list-verified': return listVerified()
  case 'publish': return publish(arg)
  default: console.log('Usage: submissions <list-pending|list-verified|publish <token>>')
}
```

**`publish <token>` logic:**

1. Read `data/submissions-verified.json`, find submission by token.
2. If not found: error "No verified submission with that token."
3. If already published: error "Already published."
4. Read `data/listings.json`.
5. Build listing record:
   ```ts
   {
     slug: tokenToSlug(submission.token),
     name: submission.name,
     website: submission.website,
     description: submission.description,
     categories: [submission.category],
     publishedAt: new Date().toISOString().slice(0, 10),
   }
   ```
6. Append to listings array, write back.
7. Mark submission as `publishedAt` in verified JSON.
8. Log: `✅ Published: "${submission.name}" → data/listings.json`

**`list-pending` output:**
```
Token              Name               Website                    Submitted
abc123-def456      My Product         https://myproduct.com      2026-04-05
```

**Add to `package.json`:**
```json
"submissions": "tsx scripts/submissions.ts"
```
