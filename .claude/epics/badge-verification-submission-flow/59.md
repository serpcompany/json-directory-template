# Task: Badge SVG template and generate-badges build script

## Context

Each site needs two static badge SVGs committed to `public/badge/`: a light and dark variant. These are served directly from the CDN and embedded by submitters on their sites. Design mirrors the nexty-monorepo template (153×44px, rounded, "FEATURED ON" + site name).

Can run in parallel with task 60 (submit API).

## Implementation

**Create `scripts/generate-badges.ts`:**

Reads all site configs from `sites/index.ts`, generates SVGs, writes to `apps/web/public/badge/`.

```ts
// For each site config:
// - Read: siteId, siteName, logo (base64 encoded PNG from assets/)
// - Write: public/badge/featured-on-{siteId}-light.svg
// - Write: public/badge/featured-on-{siteId}-dark.svg
```

**SVG template** (port from nexty-monorepo, 153×44px):

```xml
<!-- Light variant -->
<svg width="153" height="44" viewBox="0 0 153 44" xmlns="http://www.w3.org/2000/svg">
  <rect x="1" y="1" width="151" height="42" rx="5" fill="#ffffff" stroke="#e5e7eb" stroke-width="1"/>
  <image x="12" y="12" width="20" height="20" href="{logoBase64}"/>
  <text x="40" y="18" font-family="system-ui,-apple-system,sans-serif" font-size="9" font-weight="500" fill="#000000" opacity="0.8">FEATURED ON</text>
  <text x="40" y="32" font-family="system-ui,-apple-system,sans-serif" font-size="13" font-weight="700" fill="#000000">{SITE_NAME}</text>
</svg>

<!-- Dark variant: fill="#1a1a1a", stroke="#333", text fill="#ffffff"/"#eeeeee" -->
```

**Add script to `package.json`:**
```json
"generate:badges": "tsx scripts/generate-badges.ts"
```

**Hook into build**: Call `generate:badges` before `build:site` in the existing build chain.

## Files Changed

- `scripts/generate-badges.ts` (new)
- `apps/web/public/badge/*.svg` (generated, committed)
- `package.json` (new script)

## Acceptance Criteria

- [ ] `pnpm generate:badges` writes `featured-on-<site-id>-light.svg` and `featured-on-<site-id>-dark.svg` for each configured site.
- [ ] SVGs render correctly in browser (no broken image references, no external font deps).
- [ ] Light badge: white background, dark text. Dark badge: dark background, light text.
- [ ] Site name is correctly substituted from `site-config.ts`.
- [ ] Logo renders as embedded base64 (no external URL requests from the SVG).
